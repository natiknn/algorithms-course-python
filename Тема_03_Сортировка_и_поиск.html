Сортировка

Функция sorted()

sorted() возвращает новый отсортированный список. Не изменяет оригинал.

nums = [3, 1, 4, 1, 5, 9, 2]

# По умолчанию — по возрастанию
print(sorted(nums))
# [1, 1, 2, 3, 4, 5, 9]

# По убыванию
print(sorted(nums, reverse=True))
# [9, 5, 4, 3, 2, 1, 1]

# Строки — по алфавиту
words = ["banana", "apple", "cherry"]
print(sorted(words))
# ['apple', 'banana', 'cherry']

# list.sort() — сортирует на месте
nums.sort()  # меняет оригинал
print(nums)

Сортировка

Параметр key=

Параметр key принимает функцию. Для каждого элемента вызывается эта функция, и сортировка происходит по её результату.

words = ["banana", "fig", "apple", "kiwi"]

# По длине слова
print(sorted(words, key=len))
# ['fig', 'kiwi', 'apple', 'banana']

# По последней букве
print(sorted(words, key=lambda w: w[-1]))
# ['banana', 'apple', 'fig', 'kiwi']

# По числу гласных
VOWELS = set('aeiou')
def count_vowels(w):
    return sum(1 for c in w if c in VOWELS)

print(sorted(words, key=count_vowels))
# ['fig', 'kiwi', 'apple', 'banana']

Как это работает внутри?

Python как бы делает:

Словоkey(слово)
banana6
fig3
apple5
kiwi4

→ Сортирует по значениям ключей: 3,4,5,6

Важно: Функция key вызывается один раз для каждого элемента. Python не сравнивает сами элементы — он сравнивает значения ключей.
Сортировка

Лямбда-функции

Lambda — это анонимная функция (без имени) в одну строку. Удобна, когда функция простая и нужна только в одном месте.

# Обычная функция
def square(x):
    return x * x

# То же самое через lambda
square = lambda x: x * x

print(square(5))  # 25

# В sorted — можно прямо там:
nums = [(3, 'c'), (1, 'a'), (2, 'b')]

# Сортировка по первому элементу
sorted(nums, key=lambda pair: pair[0])
# [(1, 'a'), (2, 'b'), (3, 'c')]

# Сортировка по второму элементу
sorted(nums, key=lambda pair: pair[1])
# [(1, 'a'), (2, 'b'), (3, 'c')]
# Синтаксис:
# lambda аргументы: выражение

# Несколько аргументов
add = lambda x, y: x + y
print(add(3, 4))  # 7

# Для сортировки по нескольким критериям
# возвращаем КОРТЕЖ!
words = ["aaa", "bb", "cccc", "d"]

# Сначала по длине, потом по алфавиту
result = sorted(words,
    key=lambda w: (len(w), w))
print(result)
# ['d', 'bb', 'aaa', 'cccc']
Трюк: Чтобы сортировать по убыванию только по одному критерию из нескольких, поставь минус: key=lambda w: (-len(w), w) — сначала длинные, при одинаковой длине — по алфавиту.

Линейный поиск

Перебираем все элементы по одному, пока не найдём нужный. Простой, но медленный для больших данных.

Ищем значение 5 в массиве:

2
→
7
→
1
→
5
✓ Найдено! 
def linear_search(arr, target):
    for i, val in enumerate(arr):
        if val == target:
            return i  # нашли — возвращаем индекс
    return -1  # не нашли

arr = [2, 7, 1, 5, 3]
print(linear_search(arr, 5))   # 3
print(linear_search(arr, 99))  # -1

Когда использовать линейный поиск?

  • Массив маленький (до ~1000 элементов)
  • Массив не отсортирован
  • Поиск по сложному условию
  • Нужно найти первый подходящий
  • Перебор значений (например, x от 0 до 100)
Пример из задачи: Перебор значений x от 0 до 99 — это тоже линейный поиск! Проверяем каждый x, подходит ли он.

Конструкция for-else

В Python у цикла for есть необязательная ветка else. Она выполняется только если цикл завершился нормально (без break).

for элемент in последовательность:
    тело цикла
    if условие: break ← если выполнится, else пропускается
else: ← выполняется если break НЕ был вызван
    код при «не нашли»
# Поиск: нашли или нет?
target = 99
numbers = [1, 5, 7, 3, 2]

for n in numbers:
    if n == target:
        print("Нашли!")
        break
else:
    # Сюда попадаем если break не был вызван
    print("Не нашли")
    # → "Не нашли"
# Перебор значений полинома
Q = int(input())
# a0 + a1*x + ... + a5*x^5
coeffs = list(map(int, input().split()))

for x in range(100):
    value = sum(coeffs[i] * (x ** i)
                for i in range(6))
    if value == Q:
        print(x)
        break
else:
    print(-1)  # ни один x не подошёл
Без for-else пришлось бы использовать переменную-флаг. for-else — это более Pythonic способ!

Функции all() и any()

Работают с генераторами и проверяют условия для всей последовательности.

# all() — все ли элементы True?
nums = [2, 4, 6, 8]
print(all(x % 2 == 0 for x in nums))
# True — все чётные

nums2 = [2, 4, 5, 8]
print(all(x % 2 == 0 for x in nums2))
# False — 5 нечётное

# any() — хотя бы один True?
print(any(x > 7 for x in nums))
# True — 8 > 7
# Задача: является ли горка пиком?
# (строго растёт до вершины, строго убывает)

h = [1, 3, 5, 4, 2]
n = len(h)

for i in range(1, n - 1):
    # all(h[j] < h[j+1]) — строго растёт
    left_ok = all(h[j] < h[j+1]
                  for j in range(i))
    # строго убывает после вершины
    right_ok = all(h[j] > h[j+1]
                   for j in range(i, n-1))
    if left_ok and right_ok:
        print(i + 1)  # 1-индексация
        break
Важно: all() и any() работают с «ленивым» вычислением — они останавливаются как только находят ответ. Это эффективно!
Задача DEMO

Задача: Сортировка слов по числу гласных

С. Новые заклинания

Гарри выписал список потенциальных заклинаний. Нужно отсортировать их по «благозвучности»:

1) Сначала по убыванию числа гласных (a, e, i, o, u).

2) При равном числе гласных — по возрастанию длины. Сортировка должна быть стабильной.

Формат ввода

Первая строка: N (1 <= N <= 100). Далее N заклинаний — строки строчных букв, длина <= 10.

Формат вывода

Отсортированные заклинания по одному на строке.

Ввод
3
aab
aa
abababab
Вывод
abababab
aa
aab
Критерии сортировки:
  1. По числу гласных — по убыванию
  2. При равном числе гласных — по длине слова по возрастанию
VOWELS = set('aeiouAEIOU')
n = int(input())
words = [input().strip() for _ in range(n)]

def key(w):
    vowels_count = sum(1 for c in w
                       if c in VOWELS)
    # Минус — для убывания по гласным
    # len(w) — для возрастания по длине
    return (-vowels_count, len(w))

# sorted() — стабильная сортировка
result = sorted(words, key=key)

for w in result:
    print(w)

Почему кортеж?

Кортежи сравниваются поэлементно: сначала по первому, при равенстве — по второму. Именно это нам и нужно для двухуровневой сортировки.

Задача V1

Задача: Пик массива

B. Горы

Горный хребет — набор высот h₁...hₙ (n > 3). Существует «основная гора» с индексом i (1 < i < n), такая что высоты строго возрастают до hᵢ, а затем строго убывают: h₁ < h₂ < ... < hᵢ > hᵢ₊₁ > ... > hₙ.

Гарантируется, что горный хребет существует. Найдите индекс любой основной горы.

Формат ввода

Число n (3 <= n <= 10⁴), затем n чисел hᵢ (1 <= hᵢ <= 10⁹).

Формат вывода

Одно число — индекс любой основной горы (нумерация с 1).

Ввод 1
3
1 2 1
Вывод 1
2
Ввод 2
4
1 2 3 1
Вывод 2
3
Пик — позиция i, такая что все элементы слева строго возрастают до h[i], а все справа строго убывают от h[i].

Пример: [1, 3, 5, 4, 2]

1
3
5 ←пик
4
2
n = int(input())
h = list(map(int, input().split()))

for i in range(1, n - 1):
    # Проверяем: строго возрастает слева
    left_ok = all(h[j] < h[j + 1]
                  for j in range(i))
    # Проверяем: строго убывает справа
    right_ok = all(h[j] > h[j + 1]
                   for j in range(i, n - 1))

    if left_ok and right_ok:
        print(i + 1)  # 1-индексация!
        break

Почему i + 1?

В Python индексы с нуля, но задача просит позицию с единицы (1-indexed). Поэтому прибавляем 1 к найденному индексу.

Задача V3

Задача: Найти корень полинома

B. Линейный поиск по линейке

На линейке выгравирована формула для числа гвоздей Q в зависимости от длины ракеты x:

Q = a₀ + a₁·x + a₂·x² + a₃·x³ + a₄·x⁴ + a₅·x⁵

По известному Q найдите длину x (индекс деления, 0 <= x <= 99).

Формат ввода

Число Q (|Q| < 4·10²⁰), затем 6 коэффициентов aᵢ (|aᵢ| <= 10).

Формат вывода

Значение x (0 <= x <= 99) или −1, если ответа нет.

Ввод 1
6
6 5 4 3 2 1
Вывод 1
0
Ввод 2
95971372997
5 6 7 8 9 10
Вывод 2
99
Идея: Перебираем x от 0 до 99 (линейный поиск). Вычисляем значение полинома a₀ + a₁x + ... + a₅x⁵. Если равно Q — вывести x и остановиться. 
Q = int(input())
# Коэффициенты: a0, a1, ..., a5
coeffs = list(map(int, input().split()))

for x in range(100):
    # Вычисляем полином через генератор
    value = sum(coeffs[i] * (x ** i)
                for i in range(6))
    if value == Q:
        print(x)
        break   # нашли — выходим
else:
    print(-1)   # ни один x не подошёл

Это линейный поиск!

Мы перебираем все x от 0 до 99. Для каждого проверяем условие. Как только нашли подходящий — выходим. Если ни один не подошёл — else выводит -1.
