Основы

Операторы // и %

Два самых важных оператора в этой теме. Они работают с целыми числами и часто используются вместе.

//
Целочисленное деление
17 // 5 = 3

Сколько раз 5 помещается в 17?

%
Остаток от деления
17 % 5 = 2

Что осталось после деления?

✓
Проверка
3 × 5 + 2 = 17

(a//b)×b + (a%b) = a

a = 17
b = 5

print(a // b)   # 3  — целая часть от деления
print(a % b)    # 2  — остаток

# Проверка: всегда выполняется
print((a // b) * b + (a % b) == a)  # True

# Ещё примеры
print(100 // 60)   # 1  (1 час из 100 минут)
print(100 % 60)    # 40 (осталось 40 минут)
print(7 % 2)       # 1  (нечётное число → остаток 1)
print(8 % 2)       # 0  (чётное число → остаток 0)
Запомни: n % 2 == 0 — число чётное, n % 2 == 1 — нечётное. Это очень частая проверка!

Как извлечь цифры числа?

Идея: используем % 10 для получения последней цифры и // 10 для её удаления.

n = 3547
while n > 0:
    digit = n % 10  # берём цифру
    print(digit)    # 7, 4, 5, 3

while n > 0:
    total += n % 10

Системы счисления

Двоичная система счисления

В обычной (десятичной) системе 10 цифр: 0–9. В двоичной — всего 2 цифры: 0 и 1. Каждая позиция — степень числа 2.

Перевод: двоичная → десятичная

Умножаем каждый бит на соответствующую степень 2 и складываем.

Алгоритм перевода в двоичную систему

Делим число на 2 и записываем остатки снизу вверх — это и есть двоичная запись.

Трассировка: n = 13

Читаем остатки снизу вверх: 1101

def to_binary(n):
    if n == 0:
        return '0'
    bits = []
    while n > 0:
        bits.append(n % 2)  # берём бит
        n //= 2             # убираем его
    # Развернуть, т.к. биты в обратном порядке
    bits.reverse()
    return ''.join(map(str, bits))

print(to_binary(13))  # '1101'
print(to_binary(5))   # '101'
print(to_binary(8))   # '1000'
Важно: Биты накапливаются от младшего к старшему, поэтому в конце нужно перевернуть список. Последний остаток = старший бит (левый).
Тот же принцип работает для любого основания! Замените 2 на 8 — получите восьмеричную систему. На 16 — шестнадцатеричную.
Единицы измерения

Перевод единиц измерения

Задача: дано значение в мелких единицах — нужно разложить по крупным. Это делается через % и //.

Пример: секунды → ч:мин:сек

total = 3723  # секунды

sec = total % 60      # 3
total //= 60
minutes = total % 60  # 2
hours = total // 60   # 1

print(hours, minutes, sec)
# 1 2 3  →  1:02:03
Паттерн: для каждой единицы берём % коэффициент, затем // коэффициент

Задача DEMO

Задача: Старинные меры длины

А. Сажени, аршины, пяди, вершки

Древнерусская мера длины сажень состояла из трёх аршин. Один аршин делился на четыре пяди. Одна пядь состояла из 4 вершков.

Купец привёз на рынок рулон сукна длиной N вершков. Помогите ему переведите длину из вершков в сажени, аршины, пяди и вершки.

Формат ввода

Одно натуральное число N, не превосходящее 2×10⁸ — длина сукна в вершках.

Формат вывода

4 числа S A P V — сажени, аршины, пяди, вершки. A < 3, P < 4, V < 4.

Ввод
30
Вывод
0 1 3 2
Идея решения: Последовательно извлекаем каждую меру через % и уменьшаем число через //. 
n = int(input())  # общее число вершков

# Вершки — остаток от деления на 4
v = n % 4
n //= 4  # теперь n в пядях

# Пяди — остаток от деления на 4
p = n % 4
n //= 4  # теперь n в аршинах

# Аршины — остаток от деления на 3
a = n % 3
n //= 3  # теперь n в саженях

s = n    # целых саженей

print(s, a, p, v)

Пример

Входные данные: 100 вершков

  • 100 % 4 = 0 вершков, 100 // 4 = 25 пядей
  • 25 % 4 = 1 пядь, 25 // 4 = 6 аршин
  • 6 % 3 = 0 аршин, 6 // 3 = 2 сажени

Ответ: 2 0 1 0

Задача V1

Задача: Шаги к нулю в двоичной записи

А. Преврати число в ноль

Над двоичной записью числа N выполняются операции:

1) Если число заканчивается на ноль — этот ноль удаляется.

2) Если число заканчивается на единицу — из него вычитается единица.

Определите, сколько операций потребуется, чтобы превратить N в ноль.

Формат ввода

Число N (1 <= N <= 2³¹−1) в десятичной системе.

Формат вывода

Выведите количество операций.

Ввод
6
Вывод
4
Правила:
  • Если n чётное → n = n // 2 (убираем бит 0)
  • Если n нечётное → n = n - 1 (убираем бит 1)
n = int(input())
count = 0

while n > 0:
    if n % 2 == 0:  # чётное → бит 0
        n = n // 2  # убираем 0
    else:           # нечётное → бит 1
        n -= 1      # убираем 1
    count += 1

print(count)

Задача V2

Задача: Цифровой корень

С. Сумма цифр

Для числа x вычислим его сумму цифр. Если результат многозначный — снова берём сумму цифр. Повторяем до тех пор, пока не получим однозначное число.

Дано начальное число x — найдите цифру, которая получится в результате этого процесса.

Формат ввода

Одно целое число x (0 <= x <= 2³¹−1).

Формат вывода

Выведите одну цифру — ответ на задачу.

Ввод 1
2
Вывод 1
2
Ввод 2
789
Вывод 2
6
Идея: Пока число больше или равно 10, заменяем его суммой его цифр. Цифры извлекаем через % 10 и // 10. 
def get_digits_sum(number):
    """Сумма цифр числа"""
    total = 0
    while number > 0:
        total += number % 10  # берём последнюю цифру
        number //= 10         # убираем её
    return total

x = int(input())

# Продолжаем, пока x — многозначное
while x >= 10:
    x = get_digits_sum(x)

print(x)

Пример: x = 9875

  • 9+8+7+5 = 29
  • 2+9 = 11
  • 1+1 = 2 ← однозначное, стоп
