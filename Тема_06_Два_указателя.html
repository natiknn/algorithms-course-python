Два указателя

Как заменить вложенный цикл O(n²) на один проход O(n)
и три паттерна, которые решают десятки задач

Two Pointers Встречный ход Скользящее окно Префикс/Суффикс O(n)
Теория

Что такое два указателя?

Два указателя — это просто две переменные-индекса. Вместо двух вложенных циклов мы двигаем их умно и решаем задачу за один проход.

❌ Наивный подход — O(n²)

for i in range(n):
    for j in range(i+1, n):
        # проверяем пару (i, j)
        ...

Для n=10 000 → 50 000 000 итераций 😱

✅ Два указателя — O(n)

left = 0
right = n - 1
while left < right:
    # смотрим на два элемента
    # двигаем один из указателей
    ...

Для n=10 000 → максимум 10 000 итераций 🚀

💡 Когда применять: массив или строка + нужно сравнивать элементы с разных позиций + данные отсортированы или есть симметрия (палиндром, сумма).

def reverse(s):
    left, right = 0, len(s) - 1
    while left < right:
        s[left], s[right] = s[right], s[left]
        left += 1
        right -= 1

Паттерн 1 — Разбор задачи
#125 · Easy

Valid Palindrome

Условие

Строка является палиндромом, если после удаления не-буквенно-цифровых символов и приведения к нижнему регистру она читается одинаково с обеих сторон.

"A man, a plan, a canal: Panama" → True
"race a car" → False

Алгоритм

1
Ставим left=0, right=len-1
2
Пропускаем не-буквенно-цифровые с обеих сторон (isalnum())
3
Сравниваем s[left].lower() и s[right].lower()
4
Не совпали → False. Иначе двигаем оба внутрь.
def isPalindrome(s: str) -> bool:
    left = 0
    right = len(s) - 1

    while left < right:
        # пропускаем не-буквенно-цифровые
        while left < right and not s[left].isalnum():
            left += 1
        while left < right and not s[right].isalnum():
            right -= 1

        if s[left].lower() != s[right].lower():
            return False

        left += 1
        right -= 1

    return True
Время
O(n)
Память
O(1)
isalnum() → True если буква или цифра
lower() → нижний регистр

Размер окна = right - left + 1
Right всегда движется вправо (в for-цикле).
Left — только когда окно стало невалидным.

Пример: максимальная сумма подмассива длины k

def maxSum(nums, k):
    window_sum = sum(nums[:k])
    result = window_sum

    for i in range(k, len(nums)):
        # скользим: добавляем справа,
        # убираем слева
        window_sum += nums[i] - nums[i - k]
        result = max(result, window_sum)

    return result

# [2,3,4,1,5], k=3
# окна: [2,3,4]=9, [3,4,1]=8, [4,1,5]=10
# ответ: 10

Паттерн 3 — Задача

D. Удали строку — разбор решения

Напомним задачу

За операцию: выбрать непустой префикс и суффикс из одинаковых букв (не пересекаясь) и удалить. Найти минимальную длину строки.

abba → удаляем a/a → bb → удаляем b/b → 0
aboba → удаляем a/a → bob → удаляем b/b → o = 1

Почему жадно — правильно?

Если s[left] == s[right] — всегда выгодно удалить. Удаление не может навредить: оно только уменьшает строку и открывает возможности для следующего шага.

s = input().strip()
left = 0
right = len(s) - 1

while left < right and s[left] == s[right]:
    # запоминаем символ для удаления
    c = s[left]

    # сдвигаем left вправо через блок c
    while left <= right and s[left] == c:
        left += 1

    # сдвигаем right влево через блок c
    while left <= right and s[right] == c:
        right -= 1

print(max(0, right - left + 1))
⚠️ Решение не проходит все тесты! Для строки zzzaabxxxcazz даёт неверный ответ — нужно ДП. Но для большинства случаев правильно.

Массив [2,1,5,1,3,2], k=3. Найти максимальную сумму подмассива длины k.

💡 Решение 
k = 3
window = sum(a[:k])  # 2+1+5=8
best = window

    best = max(best, window)

Паттерн 1 — Two Sum
#167 · Easy

Two Sum II — встречный ход на отсортированном

Условие

Отсортированный массив numbers, найти два числа с суммой target. Вернуть индексы (нумерация с 1).

[2,7,11,15], target=9 → [1,2]

Почему работает?

  • Массив отсортирован → движение указателей предсказуемо
  • Сумма мала → двигаем left (берём большее)
  • Сумма велика → двигаем right (берём меньшее)
Трассировка [2,7,11,15], target=9:
L=0(2), R=3(15): 17>9 → R=2
L=0(2), R=2(11): 13>9 → R=1
L=0(2), R=1(7): 9==9 → [1,2] ✓ 
def twoSum(numbers, target):
    left = 0
    right = len(numbers) - 1

    while left < right:
        s = numbers[left] + numbers[right]

        if s == target:
            return [left + 1, right + 1]
        elif s < target:
            left += 1    # нужна большая сумма
        else:
            right -= 1   # нужна меньшая сумма

    return []
Время
O(n)
Память
O(1)

#3 · Medium

Longest Substring Without Repeating Characters

Условие

Найти длину самой длинной подстроки без повторяющихся символов.

"abcabcbb" → 3 ("abc")
"pwwkew" → 3 ("wke")

Алгоритм

1
Расширяем окно вправо (добавляем s[right])
2
Если новый символ уже в окне — сжимаем слева
3
Обновляем максимальную длину
Трассировка "abca":
R=0: seen={a}
R=1: seen={a,b}
R=2: seen={a,b,c}
R=3: 'a' в seen → убираем 'a' → L=1
seen={b,c,a}, max=3 
def lengthOfLongestSubstring(s):
    left = 0
    seen = set()
    result = 0

    for right in range(len(s)):
        # сжимаем пока s[right] в окне
        while s[right] in seen:
            seen.remove(s[left])
            left += 1

        seen.add(s[right])
        result = max(result, right - left + 1)

    return result
Время
O(n)
Память
O(k)

#15 · Medium

3Sum — два указателя внутри цикла

Условие

Найти все тройки [a,b,c] такие что a+b+c=0, без дубликатов.

[-1,0,1,2,-1,-4] → [[-1,-1,2],[-1,0,1]]

Идея

1
Сортируем массив
2
Фиксируем первый элемент nums[i]
3
На остатке запускаем встречный ход: ищем пару с суммой -nums[i]
def threeSum(nums):
    nums.sort()
    result = []

    for i in range(len(nums) - 2):
        if i > 0 and nums[i] == nums[i-1]:
            continue  # пропускаем дубли

        left = i + 1
        right = len(nums) - 1

        while left < right:
            s = nums[i] + nums[left] + nums[right]
            if s == 0:
                result.append(
                    [nums[i], nums[left], nums[right]])
                left += 1; right -= 1
                # пропускаем дубли
                while left < right and nums[left] == nums[left-1]:
                    left += 1
            elif s < 0:
                left += 1
            else:
                right -= 1

    return result
Время
O(n²)
Память
O(1)
