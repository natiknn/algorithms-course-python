Генераторы

Что такое генератор?

Генератор — это компактный способ создать последовательность значений прямо внутри вызова функции. Он работает как цикл, но записывается в одну строку.

Цикл (обычный способ)
numbers = [1, 2, 3, 4, 5]
squares = []
for x in numbers:
    squares.append(x * x)

print(squares)  # [1, 4, 9, 16, 25]

# Сумма через цикл
total = 0
for x in numbers:
    total += x * x
print(total)  # 55
→
Генератор (компактный способ)
numbers = [1, 2, 3, 4, 5]

# Генератор списка
squares = [x * x for x in numbers]
print(squares)  # [1, 4, 9, 16, 25]

# Генератор прямо в sum()
total = sum(x * x for x in numbers)
print(total)  # 55
Синтаксис: выражение for переменная in последовательность
Генераторы

Генератор с условием

Можно добавить if в конец генератора — тогда обрабатываются только элементы, которые проходят фильтр.

# Синтаксис с условием:
# выражение for x in seq if условие

nums = [1, 2, 3, 4, 5, 6, 7, 8]

# Только чётные числа
evens = [x for x in nums if x % 2 == 0]
print(evens)  # [2, 4, 6, 8]

# Сумма только нечётных
odd_sum = sum(x for x in nums if x % 2 != 0)
print(odd_sum)  # 16

# Количество элементов по условию
# (важный паттерн!)
count = sum(1 for x in nums if x > 5)
print(count)  # 3
# Работает и со строками!
s = "Hello, World!"

# Количество гласных
VOWELS = set('aeiouAEIOU')
cnt = sum(1 for c in s if c in VOWELS)
print(cnt)  # 3

# Только буквы в верхнем регистре
upper = [c for c in s if c.isupper()]
print(upper)  # ['H', 'W']

# Количество совпадений в двух строках
a = "abc"
b = "aec"
matches = sum(x == y for x, y in zip(a, b))
print(matches)  # 2  (a==a, c==c)
Паттерн sum(1 for x in ... if условие) — подсчитывает количество элементов, удовлетворяющих условию. Используется очень часто!
Генераторы

Генераторы: полезные функции

zip(a, b) — параллельный перебор

a = "кот"
b = "код"

for x, y in zip(a, b):
    print(x, y)
# к к
# о о
# т д

# Позиции, где символы совпадают
matches = [(x, y) for x, y in zip(a, b)
           if x == y]
print(matches)  # [('к','к'), ('о','о')]

enumerate — перебор с индексом

s = "hello"

for i, c in enumerate(s):
    print(i, c)
# 0 h
# 1 e
# 2 l
# 3 l
# 4 o

# Позиции гласных
VOWELS = set('aeiou')
positions = [i for i, c in enumerate(s)
             if c in VOWELS]
print(positions)  # [1, 4]
Важно: Генераторы внутри sum(), all(), any(), max(), min() работают без квадратных скобок — это чуть эффективнее.
Counter

collections.Counter — мультимножество

Мультимножество — как обычное множество, но каждый элемент может встречаться несколько раз. Counter автоматически подсчитывает, сколько раз встречается каждый элемент.

from collections import Counter

s = "aabbbccc"
c = Counter(s)
print(c)
# Counter({'c': 3, 'b': 3, 'a': 2})

# Доступ к количеству
print(c['a'])  # 2
print(c['z'])  # 0 (не вызывает ошибку!)

# Из списка
lst = [1, 2, 2, 3, 3, 3]
cnt = Counter(lst)
print(cnt)
# Counter({3: 3, 2: 2, 1: 1})

# Самые частые элементы
print(cnt.most_common(2))
# [(3, 3), (2, 2)]
Counter("aabbbccc")

Зачем? Быстро ответить на вопрос: «Сколько раз встречается символ X?» Используется для анаграмм, сравнения строк и других задач.
Counter

Операции над Counter (мультимножествами)

Counter поддерживает операции как у множеств: объединение, пересечение, разность.

from collections import Counter

ca = Counter("aabbc")  # {'a':2,'b':2,'c':1}
cb = Counter("abbc")   # {'a':1,'b':2,'c':1}

# Пересечение (минимум каждого)
inter = ca & cb
print(inter)
# Counter({'b': 2, 'a': 1, 'c': 1})
# Сумма: inter.values() = [2, 1, 1]

# Объединение (максимум каждого)
union = ca | cb
print(union)
# Counter({'a': 2, 'b': 2, 'c': 1})

# Разность
diff = ca - cb
print(diff)
# Counter({'a': 1})

# Сумма значений пересечения
common = sum((ca & cb).values())
print(common)  # 4

Визуально

ca = {a:2, b:2, c:1}

cb = {a:1, b:2, c:1}

Символcacbca & cb
a211
b222
c111

Сумма = 4

ca & cb берёт минимум по каждому символу — это «общие» символы строк.

Что такое палиндром?

Палиндром — строка, которая читается одинаково слева направо и справа налево.

r a c e c a r

✓ Палиндром

h e l l o

✗ Не палиндром

def is_palindrome(s):
    return s == s[::-1]
    # s[::-1] — строка наоборот

print(is_palindrome("racecar"))  # True
print(is_palindrome("hello"))    # False
print(is_palindrome("abcba"))    # True
print(is_palindrome("a"))        # True

# Без среза — через два указателя
def is_palindrome_v2(s):
    left, right = 0, len(s) - 1
    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1
    return True

Как сломать палиндром?

Если нужно сделать строку не палиндромом, изменяем один символ в левой половине:

  • Ищем первый символ в левой половине != 'a'
  • Меняем его на 'a'
  • Если все 'a' — меняем последний на 'b'

Флаг в программировании

Флаг — это переменная-булево (True/False), которая «запоминает», произошло ли некоторое событие в ходе алгоритма.

False

Событие ещё не произошло

→
True

Событие произошло!

# Флаг: нашли ли нечётное число?
numbers = [2, 4, 6, 3, 8]
found_odd = False  # флаг — пока не нашли

for n in numbers:
    if n % 2 != 0:
        found_odd = True  # подняли флаг!
        break             # можно остановиться

if found_odd:
    print("Нечётное есть!")
else:
    print("Все чётные")
# Флаг в задаче со строкой
s = list("abcba")  # палиндром
n = len(s)
changed = False  # ещё не изменяли

for i in range(n // 2):
    if s[i] != 'a':      # нашли не-'a'
        s[i] = 'a'       # меняем
        changed = True   # подняли флаг!
        break

if not changed:   # флаг не поднят?
    # Все символы левой части 'a'
    s[-1] = 'b'   # меняем последний

print(''.join(s))
Зачем флаг? Цикл заканчивается, но нам нужно знать, произошло ли что-то внутри него. Флаг сохраняет эту информацию.

Задача: Быки и Коровы

С. Быки и коровы

Вася загадал число из N цифр, Петя пытается его отгадать. Петя называет число из N цифр, а Вася отвечает, сколько «быков» и «коров».

Быки — цифры на своём месте. Коровы — цифры, которые есть в обоих числах, но на разных позициях.

Формат ввода

Два числа по одному в строке. Количество цифр <= 1000. Без ведущих нулей.

Формат вывода

В первой строке — количество «быков», во второй — «коров».

Ввод
1370
7311
Вывод
1
2
Быки — одинаковые буквы на одних позициях
Коровы — общие буквы (в любых позициях) минус быки 
from collections import Counter

a = input().strip()
b = input().strip()

# Быки: совпадения на одних позициях
bulls = sum(x == y for x, y in zip(a, b))

# Общие буквы (с учётом количества)
ca, cb = Counter(a), Counter(b)
# ca & cb — пересечение мультимножеств
common = sum((ca & cb).values())

# Коровы = общие буквы - быки
cows = common - bulls

print(bulls)
print(cows)

Задача V3

Задача: Наименьшая строка, не являющаяся палиндромом

С. Сломай палиндром

В заданной строке-палиндроме необходимо заменить ровно один символ, чтобы она перестала быть палиндромом. При этом полученная строка должна быть лексикографически минимальной.

Строка A лексикографически меньше B, если на первой различающейся позиции у A стоит меньший символ.

Формат ввода

Строка-палиндром из маленьких букв латинского алфавита, длина <= 1000.

Формат вывода

Лексикографически наименьшая не-палиндром строка. Если невозможно — пустая строка.

Ввод 1
abba
Вывод 1
aaba
Ввод 2
a
Вывод 2
(пустая строка)
Стратегия: Изменяем ровно один символ в левой половине. Это гарантирует, что строка не будет палиндромом (левая != зеркало правой). 
s = list(input().strip())
n = len(s)

if n == 1:
    print("")  # невозможно
else:
    changed = False
    # Ищем первый не-'a' в левой половине
    for i in range(n // 2):
        if s[i] != 'a':
            s[i] = 'a'       # меняем на 'a'
            changed = True   # флаг!
            break

    if not changed:
        # Все в левой половине — 'a'
        # Меняем последний символ
        s[-1] = 'b'

    print(''.join(s))

Почему именно 'a'?

Заменяем на 'a' — это лексикографически наименьший символ, что обеспечивает лексикографически наименьший результат.

Задача V3

Задача: Анаграммы

D. Анаграмма?

Задано две строки. Нужно проверить, является ли одна анаграммой другой. Анаграмма — строка, полученная из другой перестановкой букв.

Формат ввода

Две строки из строчных латинских букв, длина <= 100000. Каждая в отдельной строке.

Формат вывода

"YES" если одна строка является анаграммой другой, иначе "NO".

Ввод 1
dusty
study
Вывод 1
YES
Ввод 2
rat
bat
Вывод 2
NO
Анаграмма — строки с одинаковым набором букв (с учётом количества).

«listen» и «silent» — анаграммы! 
a = input().strip()
b = input().strip()

# Способ 1: Отсортировать и сравнить
if sorted(a) == sorted(b):
    print("YES")
else:
    print("NO")

# Способ 2: Через Counter
# from collections import Counter
# if Counter(a) == Counter(b):
#     print("YES")
# else:
#     print("NO")

Почему sorted работает?

Если строки — анаграммы, у них одинаковые символы. После сортировки оба списка будут идентичны.

«abc» → ['a','b','c']
«bca» → ['a','b','c'] ✓

Что мы изучили?

Генераторы

  • x for x in seq
  • x for x in seq if условие
  • sum(1 for ...) — счётчик
  • zip(a, b) — параллельно

Counter

  • Подсчёт символов
  • ca & cb — пересечение
  • ca | cb — объединение
  • ca - cb — разность

Палиндром и флаг

  • s == s[::-1]
  • Флаг = True/False переменная
  • Помнит «что случилось в цикле»
Главный инсайт: Генераторы + Counter превращают сложные задачи на строки в несколько лаконичных строк кода. Научитесь их читать — и большинство задач на строки станут простыми!
